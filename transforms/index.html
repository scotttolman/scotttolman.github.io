CSS Transforms

A transformation is an effect that lets an element change shape, size and position.

  • translate()

    • Translate will move an element from its current location. The method takes two parameters. The first is for the X axis (left/right). The second parameter is for the Y axis (up/down). For each parameter you state a number of units followed by the type of unit. If you want the element to move left and/or down, the value(s) will be negative. Positive values make the element move right/up.



  • rotate()

    • Rotate will rotate a div clockwise or counter-clockwise. Most of the time, the units will be specified in degrees; however, you can also use grad (400grad == 360deg), rad (3.14159265rad == 180deg), turn (1turn == 360deg). Positive numbers rotate clockwise, negative rotate counter-clockwise.



  • scale()

    • Scale changes the width and height of the element. Width and height are the two respective parameters of the scale() function. You can also pass a single parameter if the width and height are the same. The parameter(s) is a number that is multiplied by the current size. Thus, transform: scale(1) would make the element the exact same size, scale(2) would double the width and height, and scale(0.5) would be half the original size.



  • skew()

    • Skew slants the element. It takes an X and Y parameter (a number specified by degrees(deg) or similar unit). If you want to only slant the X or Y axis and not the other they use skewX() or skewY().

  • matrix()

    • Matrix combines all of the previous 2D transforms into one method. It takes 6 parameters as follows: matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY()).
