CSS Transitions

CSS transitions are similar to transforms. They both change the look of a div after a trigger sets them off. The difference is that transforms change from one form to another instantly, while transitions do so gradually. Transitions can even perform transforms in an animated or gradual way.

In order to create a transition, you need to specify in the CSS which property you want to transition as well as the length of time the transition will take. You can add multiple properties to be affected by the transition and specify each of their transition times. Also, you can specify the speed curve with the transition-timing-function property.
